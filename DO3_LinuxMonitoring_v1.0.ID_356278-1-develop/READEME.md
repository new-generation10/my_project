Linux Monitoring
Учебный DevOps/Linux-проект по разработке Bash-скриптов для мониторинга системы и исследования файловой структуры.

Что сделал:

написал Bash-скрипты для автоматизированного сбора информации о системе;
реализовал вывод сетевых, системных и дисковых параметров в удобном формате;
добавил интерактивное сохранение отчёта в .status файл;
реализовал кастомизацию цветового оформления вывода;
вынес настройки цветов в конфигурационный файл;
разработал скрипт анализа директории с выводом статистики по папкам, файлам, логам, архивам, symlink и executable-файлам;
реализовал сортировку по размеру и расчёт MD5 для исполняемых файлов;
добавил проверки входных аргументов и корректное завершение при ошибках.
