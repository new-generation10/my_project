# Linux Monitoring Toolkit

Модульный проект на Bash для исследования системы, настройки цветного вывода и анализа файловой системы в Ubuntu Server 20.04 LTS.

## Обзор проекта

В этом репозитории собран набор Bash-скриптов, решающих практические задачи администрирования и диагностики Linux:

- валидация входных параметров
- сбор системной информации
- сохранение отчёта в файл
- цветное оформление вывода в терминале
- настройка цветовой схемы через конфигурационный файл
- анализ директорий, файлов и занимаемого места

Проект разбит на несколько независимых частей, каждая из которых решает отдельную задачу.

## Что показывает этот проект

Проект демонстрирует практические навыки работы с:

- Bash scripting
- декомпозицией shell-скриптов
- валидацией пользовательского ввода
- стандартными Linux-утилитами
- ANSI-цветами в терминале
- конфигурационными файлами
- анализом файловой системы

## Технологии и инструменты

- Bash
- Ubuntu Server 20.04 LTS
- `hostname`
- `whoami`
- `ip`
- `df`
- `find`
- `du`
- `awk`
- `sort`
- `md5sum`
- `file`
